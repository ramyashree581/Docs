What is environment variable?

Environment variable: An environment variable is a dynamic "object" on a computer that stores a value.
Environment variables are, in short, variables that describe the environment in which programs run

In Windows: Go to Control panel—> Systems —> Environment variable
In Linux: export PATH = “$PATH:/usr/local/bin/python”
